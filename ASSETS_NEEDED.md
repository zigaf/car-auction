# Assets Needed

> Этот файл документирует все SVG иконки и изображения, которые нужно создать для проекта.
> Для каждого ассета указан детальный промт для генерации, путь сохранения и имя файла.

---

## ПРАВИЛО ДЛЯ РАЗРАБОТКИ

**Каждый раз, когда в коде нужна иконка или изображение — сначала добавить её в этот файл.**

Порядок работы:
1. Прежде чем использовать иконку/изображение в коде — внести запись в соответствующий раздел ниже
2. Указать точное имя файла, папку, и **детальный промт для генерации** (как будто ты заказываешь у дизайнера или генерируешь через AI)
3. В коде ссылаться на путь `assets/icons/имя-файла.svg` (или images/, brand-icons/)
4. После того как файл будет создан и сохранён — отметить строку ✅

## Как писать промт

Промт должен быть настолько детальным, чтобы любой дизайнер или AI-генератор создал именно то, что нужно. Включать:

- **Что изображено** — конкретный объект, не абстракция
- **Стиль** — outline/filled/flat/isometric/3d, stroke-width
- **Цвет** — currentColor / монохром / конкретный HEX
- **Формат** — SVG, viewBox, размер
- **Детали** — углы, пропорции, толщина линий, характерные черты
- **Чего НЕ должно быть** — градиентов, теней, лишних деталей

**Пример хорошего промта:**
> `SVG icon, outline style, stroke-width 1.5, viewBox 0 0 24 24, currentColor, no fill. A car silhouette viewed from the side: sedan body, two visible wheels as circles, simple door lines, no interior details. Clean minimal lines, geometric proportions. No gradients, no shadows, no text.`
